CARBOPLATIN-INJECTION-MG/ML (卡鉑注射液)

CARBOPLATIN-INJECTION-MG/ML (卡鉑注射液) 是一種含有卡鉑的注射劑型藥物。它是一種用於治療多種癌症的化學藥物。

用途

CARBOPLATIN-INJECTION-MG/ML (卡鉑注射液) 主要用於治療以下類型的癌症：

  • 卵巢癌
  • 肺癌
  • 頭頸部癌症
  • 膀胱癌
  • 子宮內膜癌
  • 睪丸癌

使用方法

CARBOPLATIN-INJECTION-MG/ML (卡鉑注射液) 通過靜脈注射給藥。劑量和治療方案會根據患者的具體情況而定，如年齡、身體狀況和癌症類型。

副作用

CARBOPLATIN-INJECTION-MG/ML (卡鉑注射液) 的常見副作用包括：

  • 噁心和嘔吐
  • 骨髓抑制（造血功能受損）
  • 腎功能損害
  • 神經系統毒性（如感覺異常或運動障礙）
    請注意，這只是一些常見副作用，具體副作用可能因個體差異而有所不同。

注意事項

在使用 CARBOPLATIN-INJECTION-MG/ML (卡鉑注射液) 期間，請注意以下事項：

  • 定期進行血液檢查以監測藥物對造血系統的影響。
  • 與醫生討論可能的副作用和治療期間的注意事項。
  • 通知醫生關於任何已經使用的藥物、過敏反應或其他健康問題。

VITRAKVI mg 膠囊

VITRAKVI 是一種口服藥物，以膠囊形式提供。它的主要成分是 Larotrectinib (LOXO-101)。VITRAKVI 被用於治療一種罕見的癌症類型，稱為TRK基因融合陽性腫瘤。

藥物分類

  • 抗癌藥物
  • 靶向治療藥物

作用機制

VITRAKVI 的作用機制與 Larotrectinib 相同。它是一種TRK（神經生長因子受體）抑制劑，通過抑制TRK基因融合產生的異常蛋白質，阻斷了腫瘤細胞的生長和擴散。

適應症

VITRAKVI 適用於具有TRK基因融合的成人和兒童患者，無論其癌症類型。這種藥物已被批准用於治療局部晚期或轉移性TRK基因融合陽性腫瘤。

用法和劑量

  • VITRAKVI 以口服藥片形式供應。
  • 劑量根據患者的體重和身體狀況進行個體化調整。
  • 患者應按照醫生的指示準確服用藥物。

不良反應

VITRAKVI 的常見不良反應包括：

  • 疲勞
  • 噁心
  • 嘔吐
  • 腹瀉
  • 頭暈
  • 食慾不振

注意事項

  • 在使用 VITRAKVI 之前，告知醫生您的藥物過敏史和現有的健康狀況。
  • 如果出現嚴重不良反應或過敏反應，請立即就醫。
  • 與醫生保持密切聯繫，定期進行藥物療效和安全性的評估。

Farydak--mg-capsules介紹

Created2023-10-25Updated2024-01-30

Farydak（法立達克）- mg膠囊

簡介

法立達克（Farydak）是一種口服藥物，以膠囊形式提供。它含有活性成分泛歐必納斯塔（Panobinostat），屬於組蛋白去乙醯化酶抑制劑（HDAC抑制劑）。法立達克被用於治療多發性骨髓瘤（MM）的成人患者，特別是對於已經接受過至少兩種治療方案但無效的患者。

作用機制

法立達克中的泛歐必納斯塔可以抑制組蛋白去乙醯化酶（HDAC），這種酶在癌細胞中過度活躍。通過抑制HDAC，它可以阻止癌細胞中某些基因的表達，從而抑制腫瘤的生長和擴散。

使用方法

法立達克以口服膠囊的形式給予。劑量和用法應根據醫生的指示進行調整。通常，每天服用一次，連續服用數天，然後停藥一段時間，以便身體恢復。

副作用

法立達克可能引起一些副作用，包括疲勞、噁心、嘔吐、腹瀉、食慾不振、頭痛、失眠等。嚴重的副作用可能包括心律不整、出血、感染等。如果出現嚴重的副作用，應立即告知醫生。

注意事項

在使用法立達剋期間，患者應定期進行血液檢查，以監測藥物的效果和可能的副作用。同時，患者應避免使用其他藥物或草藥，除非經過醫生同意。

VINBLASTINE-SULPHATE-FOR-INJECTION-DBL介紹

Created2023-10-08Updated2024-01-30

VINBLASTINE-SULPHATE-FOR-INJECTION-DBL (硫酸長春鹼注射劑)

1. 什麼是VINBLASTINE-SULPHATE-FOR-INJECTION-DBL？

VINBLASTINE-SULPHATE-FOR-INJECTION-DBL是一種含有硫酸長春鹼的注射劑。它是一種用於治療癌症的藥物。

2. VINBLASTINE-SULPHATE-FOR-INJECTION-DBL有哪些用途？

VINBLASTINE-SULPHATE-FOR-INJECTION-DBL主要用於治療多種類型的癌症，包括淋巴瘤、卵巢癌、乳腺癌、肺癌等。它可以通過抑制癌細胞的生長和分裂來減緩癌症的發展。

3. VINBLASTINE-SULPHATE-FOR-INJECTION-DBL如何使用？

VINBLASTINE-SULPHATE-FOR-INJECTION-DBL通常以注射液的形式給予患者。具體的劑量和使用方法應由醫生根據患者的情況來確定。治療期間應嚴格遵循醫生的指示。

4. VINBLASTINE-SULPHATE-FOR-INJECTION-DBL有哪些副作用？

使用VINBLASTINE-SULPHATE-FOR-INJECTION-DBL可能會引起一些副作用，包括噁心、嘔吐、脫髮、骨髓抑制等。某些患者可能還會出現感染、出血傾向、神經病變等嚴重副作用。如果出現任何不適或副作用，應立即告知醫生。

5. 是否存在注意事項或禁忌症？

VINBLASTINE-SULPHATE-FOR-INJECTION-DBL在某些情況下可能會有禁忌症，例如妊娠期間、哺乳期、嚴重的肝功能障礙等。在使用VINBLASTINE-SULPHATE-FOR-INJECTION-DBL之前，應告知醫生有關自己的過敏史、其他用藥情況以及已有的健康問題。

Tamos-Capsules--mg介紹

Created2023-09-12Updated2024-01-30

Tamos-Capsules (mg)

介紹

Tamos-Capsules 是一種藥物，其主要成分是鹽酸替莫唑胺（Temozolomide）。它屬於一種化療藥物，用於治療某些類型的腦腫瘤，特別是惡性膠質瘤。

作用原理

Tamos-Capsules 的作用原理與 Temozolomide 相同。它通過抑制腫瘤細胞中的DNA修復機制，從而阻止腫瘤細胞的增殖。它可以對DNA中的嘌呤鹼基進行甲基化，進而引起DNA損傷和細胞凋亡。

使用方法

Tamos-Capsules 是以口服膠囊的形式使用。每個膠囊中含有特定劑量的鹽酸替莫唑胺。具體的劑量和療程應根據醫生的指示和患者的具體情況而定。請務必按照醫生的建議使用藥物。

副作用

Tamos-Capsules 的副作用與 Temozolomide 相似。常見的副作用包括噁心、嘔吐、疲勞、食慾不振和頭痛等。它也可能導致白血球減少、血小板減少和貧血等血液問題。某些患者可能還會出現皮膚反應、肝功能異常和腎功能異常等不良反應。

注意事項

在使用 Tamos-Capsules 期間，患者應該定期進行血液檢查，以監測血細胞計數和肝功能。同時，患者應避免接觸妊娠中的女性，因為 Tamos-Capsules 可能對胎兒造成傷害。
請注意，以上僅為一般信息，具體的用藥指導和注意事項應遵循醫生的建議。
