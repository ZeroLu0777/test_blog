CARBOPLATIN-INJECTION-DBL (卡鉑注射液)

CARBOPLATIN-INJECTION-DBL (卡鉑注射液) 是一種含有卡鉑成分的注射液，用於治療多種癌症。它屬於鉑類藥物，能夠阻止癌細胞的生長和分裂。

用途

CARBOPLATIN-INJECTION-DBL (卡鉑注射液) 主要用於治療以下類型的癌症：

  • 卵巢癌
  • 肺癌
  • 頭頸部癌症
  • 膀胱癌
  • 子宮內膜癌
  • 睪丸癌

使用方法

CARBOPLATIN-INJECTION-DBL (卡鉑注射液) 通過靜脈注射給藥。劑量和治療方案會根據患者的具體情況而定，如年齡、身體狀況和癌症類型。

副作用

CARBOPLATIN-INJECTION-DBL (卡鉑注射液) 的常見副作用包括：

  • 噁心和嘔吐
  • 骨髓抑制（造血功能受損）
  • 腎功能損害
  • 神經系統毒性（如感覺異常或運動障礙）
    請注意，這只是一些常見副作用，具體副作用可能因個體差異而有所不同。

注意事項

在使用 CARBOPLATIN-INJECTION-DBL (卡鉑注射液) 期間，請注意以下事項：

  • 定期進行血液檢查以監測藥物對造血系統的影響。
  • 與醫生討論可能的副作用和治療期間的注意事項。
  • 通知醫生關於任何已經使用的藥物、過敏反應或其他健康問題。

PARAPLATIN (卡鉑注射液)

PARAPLATIN (卡鉑注射液) 是一種用於靜脈注射的化學藥物。它含有活性成分卡鉑，屬於鉑類藥物，用於治療多種癌症。

用途

PARAPLATIN (卡鉑注射液) 主要用於治療以下類型的癌症：

  • 卵巢癌
  • 膀胱癌
  • 卵巢上皮癌
  • 卵巢生殖細胞腫瘤
  • 腺癌
  • 頭頸部癌症

使用方法

PARAPLATIN (卡鉑注射液) 是一種靜脈注射藥物，由醫護人員在醫療設施中給予。劑量和治療方案會根據患者的具體情況而定，如年齡、身體狀況和癌症類型。

副作用

使用 PARAPLATIN (卡鉑注射液) 可能會引起一些副作用，包括：

  • 噁心和嘔吐
  • 骨髓抑制（造血功能受損）
  • 腎功能損害
  • 神經系統毒性（如感覺異常或運動障礙）
    請注意，這只是一些常見副作用，具體副作用可能因個體差異而有所不同。

注意事項

在使用 PARAPLATIN (卡鉑注射液) 期間，請注意以下事項：

  • 只能在醫療設施中由專業醫護人員給予。
  • 定期進行血液檢查以監測藥物對造血系統的影響。
  • 與醫生討論可能的副作用和治療期間的注意事項。
  • 通知醫生關於任何已經使用的藥物、過敏反應或其他健康問題。

Panobinostat (泛歐必納斯塔)

簡介

泛歐必納斯塔（Panobinostat）是一種抗癌藥物，屬於組蛋白去乙醯化酶抑制劑（HDAC抑制劑）。它被用於治療多發性骨髓瘤（MM）的成人患者，特別是對於已經接受過至少兩種治療方案但無效的患者。

作用機制

泛歐必納斯塔可以抑制組蛋白去乙醯化酶（HDAC），這種酶在癌細胞中過度活躍。通過抑制HDAC，它可以阻止癌細胞中某些基因的表達，從而抑制腫瘤的生長和擴散。

使用方法

泛歐必納斯塔以口服藥片的形式給予。劑量和用法應根據醫生的指示進行調整。通常，每天服用一次，連續服用數天，然後停藥一段時間，以便身體恢復。

副作用

泛歐必納斯塔可能引起一些副作用，包括疲勞、噁心、嘔吐、腹瀉、食慾不振、頭痛、失眠等。嚴重的副作用可能包括心律不整、出血、感染等。如果出現嚴重的副作用，應立即告知醫生。

注意事項

在使用泛歐必納斯塔期間，患者應定期進行血液檢查，以監測藥物的效果和可能的副作用。同時，患者應避免使用其他藥物或草藥，除非經過醫生同意。

Mekinist 膜衣片 (mg)

簡介

Mekinist 膜衣片是一種口服藥物，含有曲美替尼（Trametinib）成分。曲美替尼屬於酪氨酸激酶抑制劑，用於治療某些類型的癌症，特別是具有BRAF V600E或BRAF V600K突變的非小細胞肺癌和黑色素瘤。
## 作用機制
Mekinist 膜衣片中的曲美替尼可以抑制BRAF蛋白激酶的活性，從而阻斷細胞增殖和生長的信號傳遞路徑。這有助於抑制腫瘤的生長和擴散。
## 使用方法
Mekinist 膜衣片以口服藥片的形式給予，通常每天一次。用藥前應詳細閱讀並遵從醫生或藥劑師的指示。劑量和療程的長度會根據患者的具體情況而定。
## 常見副作用
Mekinist 膜衣片的常見副作用包括疲勞、發燒、皮膚紅斑、皮疹、腹瀉、噁心和嘔吐等。如果出現嚴重的副作用，應立即告訴醫生。
## 注意事項
- 在使用 Mekinist 膜衣片期間，應定期進行醫學檢查和血液檢測，以監測治療的效果和可能的副作用。
- Mekinist 膜衣片可能與其他藥物相互作用，因此在使用其他藥物之前應先告知醫生或藥劑師。
- 孕婦和哺乳期婦女應避免使用 Mekinist 膜衣片，因為它可能對胎兒或嬰兒產生不良影響。
以上是對 Mekinist 膜衣片的簡要介紹，如果您有更多問題或需要進一步的資訊，請諮詢您的醫生或藥劑師。
注意：本回答僅供參考，請在使用任何藥物之前諮詢專業醫療人員。

腦癌介紹

Created2023-06-13Updated2024-01-30

腦癌教育資訊

腦部功能介紹

腦部是我們身體最重要的器官之一，負責控制我們的思考、運動、感覺和語言能力。它由許多不同的部分組成，包括大腦、小腦和腦幹。這些部分協同工作，使我們能夠正常運作。

腦癌的常見原因

腦癌的具體原因尚不完全清楚，但以下因素可能與其發生有關：

  • 基因突變：某些基因突變可能會導致細胞異常增殖，最終形成腦腫瘤。
  • 遺傳因素：某些家族中存在腦癌的遺傳風險。
  • 輻射暴露：長期接觸高劑量的輻射可能增加罹患腦癌的風險。
  • 免疫系統問題：免疫系統功能異常可能增加腦癌的發生機率。

腦癌的常見症狀

腦癌的症狀可能因腫瘤位置和大小而有所不同，但以下是一些常見的症狀：

  • 頭痛：頭痛可能是最常見的腦癌症狀之一，特別是早上醒來時。
  • 噁心和嘔吐：腦癌可能導致噁心和嘔吐，尤其是在早期。
  • 視力問題：視力模糊、雙視或視野缺失可能與腦癌有關。
  • 肌肉無力或協調困難：腦癌可能導致肌肉無力、協調困難或行走困難。
  • 言語和認知問題：腦癌可能影響語言和認知能力，例如記憶力下降或言語困難。

腦癌的常見檢查

根據NCCN（National Comprehensive Cancer Network）的治療指南，腦癌的常見檢查包括：

  • 神經學檢查：醫生會評估患者的神經系統功能，包括感覺、運動和協調能力。
  • 腦部影像學檢查：這包括MRI（磁共振成像）和CT（電腦斷層掃描）等檢查，以檢視腦部結構和腫瘤位置。
  • 腦脊液檢查：醫生可能會提取腦脊液樣本進行檢查，以確定是否存在腫瘤細胞。
  • 生物檢體檢查：透過手術或穿刺樣本，醫生可以檢測腫瘤組織的細胞學和分子遺傳學特徵。

腦癌的疾病分期和常用藥物

疾病分期

根據NCCN（National Comprehensive Cancer Network）的治療指南，腦癌的分期通常使用TNM系統來描述腫瘤的大小、深度和擴散情況。以下是腦癌的常見分期：

  • Stage I：腫瘤侷限在腦部的一個區域，尚未擴散到其他組織。
  • Stage II：腫瘤已經擴散到腦部的其他區域，但尚未達到重要結構。
  • Stage III：腫瘤已經擴散到腦部的重要結構，如腦幹。
  • Stage IV：腫瘤已經擴散到腦部以外的其他部位，如脊髓或其他器官。
    疾病分期有助於醫生制定適當的治療計劃和預測預後。

常用藥物

根據NCCN的治療指南，腦癌的治療通常包括以下常用藥物：

  1. 帕尼妥（Pemetrexed）：一種化學治療藥物，可抑制腫瘤細胞的生長和擴散。
  2. 氟尿嘧啶（Fluorouracil）：一種抗代謝藥物，可阻斷腫瘤細胞的DNA合成，抑制其增殖。
  3. 卡培他賓（Carboplatin）：一種鉑化合物，可幹擾腫瘤細胞的DNA修復機制，使其死亡。
  4. 替諾福韋（Temozolomide）：一種鹽酸鹽類藥物，可抑制腫瘤細胞的DNA修復能力，阻止其增殖。
    這些藥物通常作為腦癌的化學治療或放射治療的一部分，醫生會根據患者的具體情況和腫瘤特徵來選擇最適合的治療方案。

腦癌患者的自我照顧指南

根據HOPE Foundation for Cancerfree Care的健康教育手冊，以下是一些腦癌患者可以遵循的自我照顧指南：

1. 遵從醫療專業人員的建議

  • 定期就診：按照醫生的建議定期進行檢查和隨訪，以監測腫瘤的狀態和治療效果。
  • 藥物管理：按照醫生處方的藥物劑量和時間表，準時服藥，不要隨意停藥或更改劑量。
  • 注意副作用：瞭解可能的治療副作用，如噁心、嘔吐、疲勞等，及時向醫生報告。

2. 健康生活方式

  • 飲食均衡：選擇營養豐富的食物，包括蔬菜、水果、全穀物和蛋白質，避免高脂肪和高糖飲食。
  • 適度運動：根據醫生的建議進行適度的身體活動，如散步、瑜伽或輕度運動，有助於維持身體健康和心理平衡。
  • 戒煙限酒：避免吸煙和過量飲酒，這些習慣可能增加腦癌的風險並幹擾治療效果。

3. 心理和情緒支持

  • 尋求支持：與家人、朋友或腦癌支持團體保持聯繫，分享感受和經驗，獲得情緒上的支持。
  • 心理輔導：考慮尋求專業心理輔導，幫助處理壓力、焦慮和情緒困擾。
  • 放鬆技巧：嘗試放鬆技巧，如深呼吸、冥想或瑜伽，有助於減輕壓力和提升心理健康。

4. 安全和預防

  • 頭部保護：避免頭部受傷，戴上安全帽或頭盔，特別是參與高風險活動。
  • 防曬措施：在陽光強烈的時候，使用適當的防曬霜、帽子和太陽眼鏡，保護頭部免受紫外線的損害。

Temodal Powder for Solution for Infusion

介紹

Temodal Powder for Solution for Infusion 是一種藥物，用於靜脈注射治療特定類型的腦腫瘤。它含有一種名為替莫唑胺的活性成分。

作用原理

Temodal Powder for Solution for Infusion 的作用原理與口服藥物 Temozolomide 相似。它通過抑制腫瘤細胞中的DNA修復機制，從而阻止腫瘤細胞的增殖。它可以對DNA中的嘌呤鹼基進行甲基化，進而引起DNA損傷和細胞凋亡。

使用方法

Temodal Powder for Solution for Infusion 是以靜脈注射的方式給予。它需要在醫療專業人員的監督下使用，並按照醫生的指示進行劑量和輸注速度的調整。

副作用

Temodal Powder for Solution for Infusion 的常見副作用包括噁心、嘔吐、疲勞、食慾不振和頭痛等。它也可能導致白血球減少、血小板減少和貧血等血液問題。某些患者可能還會出現皮膚反應、肝功能異常和腎功能異常等不良反應。

注意事項

在使用 Temodal Powder for Solution for Infusion 期間，患者應該定期進行血液檢查，以監測血細胞計數和肝功能。同時，患者應避免接觸妊娠中的女性，因為 Temodal Powder for Solution for Infusion 可能對胎兒造成傷害。
請注意，以上僅為一般信息，具體的用藥指導和注意事項應遵循醫生的建議。

VITRAKVI (mg/ml) 口服溶液

VITRAKVI (mg/ml) 口服溶液是一種藥物，用於治療TRK基因融合陽性腫瘤。以下是有關 VITRAKVI 口服溶液的介紹：

藥物分類

  • 抗癌藥物
  • 靶向治療藥物

作用機制

VITRAKVI 是一種TRK（神經生長因子受體）抑制劑。它能夠抑制TRK基因融合產生的異常蛋白質，從而阻斷腫瘤細胞的生長和擴散。

適應症

VITRAKVI 口服溶液適用於具有TRK基因融合的成人和兒童患者，無論其癌症類型。這種藥物已被批准用於治療局部晚期或轉移性TRK基因融合陽性腫瘤。

用法和劑量

  • VITRAKVI 口服溶液以口服形式給予。
  • 劑量根據患者的體重和身體狀況進行個體化調整。
  • 患者應按照醫生的指示準確服用藥物。

不良反應

VITRAKVI 口服溶液的常見不良反應包括：

  • 疲勞
  • 噁心
  • 嘔吐
  • 腹瀉
  • 頭暈
  • 食慾不振

注意事項

  • 在使用 VITRAKVI 口服溶液之前，告知醫生您的藥物過敏史和現有的健康狀況。
  • 如果出現嚴重不良反應或過敏反應，請立即就醫。
  • 與醫生保持密切聯繫，定期進行藥物療效和安全性的評估。
