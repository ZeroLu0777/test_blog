腦癌教育資訊

腦部功能介紹

腦部是人體最重要的器官之一，負責控制身體的各種功能和思維過程。腦部由不同的區域組成，每個區域負責不同的功能，包括：

  • 大腦皮質：負責思考、學習、記憶和語言等高級功能。
  • 小腦：協調運動和平衡。
  • 腦幹：控制基本的生理功能，如呼吸、心跳和消化。

腦癌的常見原因

腦癌的具體原因尚不清楚，但以下因素可能增加罹患腦癌的風險：

  • 遺傳因素：某些基因突變可能與腦癌有關。
  • 曾接受過頭部輻射治療。
  • 免疫系統問題。
  • 某些遺傳疾病。

腦癌的常見症狀

腦癌的症狀可能因腫瘤的位置和大小而有所不同。以下是一些常見的症狀：

  • 頭痛：可能是頻繁且劇烈的頭痛，不易緩解。
  • 噁心和嘔吐：特別是早上或在特定位置。
  • 視力問題：包括模糊視力、視力喪失或雙視。
  • 肢體無力或麻木感。
  • 言語和認知問題：困難發音、記憶力下降等。

腦癌的常見檢查

根據NCCN（National Comprehensive Cancer Network）的治療指南，腦癌的常見檢查包括：

  • 神經學檢查：醫生會評估病人的神經系統功能，包括運動、感覺和協調能力等。
  • 腦部影像學檢查：如頭部MRI（磁共振成像）或CT（電腦斷層掃描）等，用於檢測和定位腫瘤。
  • 腦脊液檢查：通過腰椎穿刺檢測腦脊液中的腫瘤標記物。

腦癌的疾病分期和常用藥物

腦癌的疾病分期

腦癌的分期是根據腫瘤的大小、擴散程度和淋巴結參與情況來進行評估。根據NCCN治療指南，腦癌的分期通常使用TNM分期系統，其中包括以下要素：

  1. T分期（腫瘤大小）：
    • T1：腫瘤直徑小於2.5釐米。
    • T2：腫瘤直徑介於2.5至5釐米之間。
    • T3：腫瘤直徑大於5釐米。
  2. N分期（淋巴結參與情況）：
    • N0：無淋巴結轉移。
    • N1：有淋巴結轉移。
  3. M分期（遠處轉移）：
    • M0：無遠處轉移。
    • M1：有遠處轉移。
      根據這些分期要素的組合，醫生可以確定腦癌的疾病分期，從而制定適當的治療計劃。

腦癌的常用藥物

根據NCCN治療指南，腦癌的治療通常包括以下常用藥物：

  1. 帕尼妥珠單抗（Bevacizumab）：一種抗血管生成藥物，用於抑制腫瘤的血管形成，減少腫瘤的營養供應。
  2. 奎尼替尼（Lomustine）：一種化療藥物，用於抑制腫瘤細胞的生長和分裂。
  3. 氟脫氧尿苷（Temozolomide）：一種化療藥物，用於幹擾腫瘤細胞的DNA修復機制，從而抑制腫瘤生長。
  4. 奧沙利鈉（Osimertinib）：一種標靶治療藥物，用於治療具有EGFR突變的腦癌。
    這些藥物的使用與劑量會根據患者的具體情況和疾病分期進行個別評估和調整。

腦癌患者自我照顧指南

根據HOPE Foundation for Cancerfree Care（希望基金會）的健康教育手冊，以下是腦癌患者自我照顧的指南：

身體健康照顧

  • 遵從醫生的治療計劃：按時服藥，接受放射治療或化療等治療方式。
  • 注意飲食均衡：食用新鮮水果、蔬菜、全穀物和富含蛋白質的食物，避免高糖和高脂肪食品。
  • 確保充足休息：保持良好的睡眠品質，適當休息有助於身體康復。
  • 適度運動：根據醫生的建議，進行適量的運動，如散步、瑜伽或輕度運動。
  • 注意個人衛生：保持良好的衛生習慣，勤洗手，避免感染。

心理健康照顧

  • 尋求支持：與家人、朋友或支持團體交流，分享自己的感受和困擾。
  • 接受心理輔導：如有需要，尋求專業心理輔導或心理治療，幫助應對情緒困擾和壓力。
  • 培養興趣愛好：參與自己喜歡的活動，如閱讀、聽音樂、繪畫等，有助於舒緩壓力和提升心情。
  • 學習放鬆技巧：嘗試深呼吸、冥想或瑜伽等放鬆技巧，幫助減輕焦慮和壓力。

社交支持和資源

  • 建立支持網絡：與其他腦癌患者或康復者聯繫，分享經驗和資源。
  • 尋求社會資源：瞭解當地的癌症支持組織、慈善機構和相關資源，以獲得更多支持和幫助。
  • 與醫療團隊合作：與醫生、護士和其他醫療專業人員建立良好的溝通和合作關係，獲得最佳的醫療照護。

PARAPLATIN IV注射劑

PARAPLATIN-IV-INJECTION 是一種含有 Carboplatin 的靜脈注射劑。它是一種常用的化療藥物，用於治療多種癌症，特別是卵巢癌和肺癌。

藥理學

PARAPLATIN-IV-INJECTION 的主要成分是 Carboplatin。Carboplatin 通過與DNA結合，幹擾癌細胞的DNA複製和修復過程，從而抑制癌細胞的生長和分裂。它主要通過形成DNA加合物，阻礙DNA鏈的延伸，導致癌細胞死亡。

適應症

PARAPLATIN-IV-INJECTION 主要用於以下癌症的治療：

  • 卵巢癌
  • 肺癌
  • 卵巢上皮癌
  • 卵巢生殖細胞腫瘤
  • 腺樣囊性癌
  • 胸腺瘤

使用方法

PARAPLATIN-IV-INJECTION 通常通過靜脈注射給藥。劑量會根據患者的體重、身體狀況和癌症類型進行調整。治療方案通常由醫生根據具體情況制定。

副作用

PARAPLATIN-IV-INJECTION 的常見副作用包括：

  • 骨髓抑制：可能導致貧血、白細胞減少和血小板減少。
  • 噁心和嘔吐
  • 脫髮
  • 腎功能損害
  • 神經毒性：可能導致感覺異常和周圍神經病變。
    請注意，這只是一份簡要介紹，具體的劑量和用法應由醫生根據患者的情況決定。

Larotrectinib (LOXO-101)

Larotrectinib（LOXO-101）是一種靶向性抗癌藥物，用於治療特定類型的癌症。它屬於一類稱為TRK（神經營養因子受體）抑制劑的藥物，可以抑制異常活躍的TRK蛋白。TRK蛋白在正常情況下參與神經細胞的生長和發育，但在某些癌症中，TRK蛋白會異常活躍，導致腫瘤的生長和擴散。
Larotrectinib是一種口服藥物，很容易被身體吸收。它可以通過抑制異常活躍的TRK蛋白來阻斷腫瘤的生長。Larotrectinib被廣泛用於治療TRK融合陽性的癌症，包括兒童和成人患者。
以下是一些關於Larotrectinib的常見問題和答案：
1. Larotrectinib的適應症是什麼？
Larotrectinib適用於TRK融合陽性的癌症患者，包括兒童和成人。TRK融合是指TRK基因與其他基因融合，導致TRK蛋白異常活躍。
2. Larotrectinib如何使用？
Larotrectinib是一種口服藥物，按照醫生的指示服用。劑量和用法會根據患者的具體情況而定。
3. Larotrectinib有哪些副作用？
Larotrectinib的常見副作用包括疲勞、噁心、嘔吐、腹瀉、頭痛和肌肉疼痛。如果出現嚴重的副作用或不適，應立即告知醫生。
4. Larotrectinib是否與其他藥物相互作用？
Larotrectinib可能與其他藥物相互作用，包括其他藥物、補充劑和草藥。在使用Larotrectinib之前，應告知醫生正在使用的所有藥物。
5. Larotrectinib的療效如何？
Larotrectinib在TRK融合陽性的癌症患者中顯示出很好的療效。它可以抑制腫瘤的生長和擴散，並提高患者的生存率。
以上是關於Larotrectinib（LOXO-101）的簡介和常見問題的回答。

Farydak（法利達克）- mg膠囊

簡介

Farydak（法利達克）是一種口服藥物，屬於組蛋白去乙醯化酶抑制劑（HDAC抑制劑）的一種。它的主要成分是Panobinostat（帕諾布尼斯塔）。

用途

Farydak（法利達克）主要用於治療多發性骨髓瘤（MM），一種影響骨髓的癌症。它通常與其他抗癌藥物（如bortezomib和dexamethasone）一起使用，以提高治療效果。

作用機制

Farydak（法利達克）的主要成分Panobinostat（帕諾布尼斯塔）是一種組蛋白去乙醯化酶（HDAC）抑制劑。它可以抑制HDAC的活性，增加組蛋白的乙醯化水平，從而改變基因的表達。這有助於抑制癌細胞的生長和增殖。

使用方法

  • Farydak（法利達克）以膠囊形式給予。
  • 治療方案和劑量會根據患者的具體情況而定，應遵從醫生的指示。
  • 藥物應整粒吞服，不要咀嚼、破碎或打碎。

常見副作用

使用Farydak（法利達克）可能會引起以下常見副作用：

  • 噁心、嘔吐、腹瀉
  • 疲勞、虛弱感
  • 食慾不振
  • 頭痛
  • 體重減輕
